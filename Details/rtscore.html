About rts-core project

Source code and more details available on Bitbucket project

A game where player controls the base increases his army and plays against bots with different behaviors. Base contains buildings, resources, and an army. The action takes place on the 15x15-30x30 map where each player has 5 buildings at his disposal and can improve them. During the game you can increase your base by capturing free space on the map. Each player can attack any of enemy bases. Bots are not always aggressive to the player. When a player attacks he chooses the number and type of units with which he attacks. On a successful attack the enemy base is destroyed and the attacker receives a reward. Otherwise attack squad is dies.

Features

  • Lightweight UI-independent and testable game core
  • General components are covered with tests

In-Game features

Buildings

  • Living module - increases the limit of people at the base.
  • Barrack - allows to train units.
  • Workshop - produces goods per game tick.
  • Portal - allows you to buy / sell goods.
  • Wall - increase the protection of units at the base.

Resources

  • People - base population. 200 by default and +100 by each game update.
  • Credits - money to buy goods or to train units. 300 by default and +100 by each game update.
  • Goods - products that are manufactured at the base. 500 by default and +10% from people count by each game update.
